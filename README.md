## Featured projects

### Basketball Analytics Platform — *Granted Patent*
> Real-time player tracking & performance analytics from match footage

- **89% detection accuracy** on 4,500+ manually annotated images — YOLOv11 with hyperparameter tuning
- ByteTrack + OSNet Re-ID for stable multi-player tracking across frames
- Pixel-to-metre speed conversion, movement heatmaps, 30 FPS Streamlit dashboard

`YOLOv11` `ByteTrack` `OSNet` `OpenCV` `PyTorch` `Streamlit`

---

### Generative AI Lawyer — Indian Penal Code
> Domain-specific LLM fine-tuned on Indian legal datasets

- **75% GPU memory reduction** via 4-bit QLoRA quantisation
- LoRA (r=8, alpha=64) fine-tuning with gradient accumulation for low-resource training
- Custom tokenisation pipeline, real-time legal Q&A and summarisation via Streamlit

`LLaMA-2-7B` `QLoRA` `LoRA` `PEFT` `HuggingFace` `Streamlit`

---

### Legal Document RAG System
> Production-grade 8-layer RAG pipeline with CI evaluation gate

- SemanticChunker → OpenAI embeddings → Qdrant vector store
- BM25+vector hybrid retrieval → Cohere reranking → RAGAS evaluation
- Automated CI eval gate via GitHub Actions — CourtListener open case data

`LangChain` `Qdrant` `RAGAS` `Cohere` `FastAPI` `GitHub Actions` `Docker`
